# resumo do primeiro módulo de AZ - 900 - Microsoft

# O QUE É COMPUTAÇÃO EM NUVEM ?
* A COMPUTAÇÃO EM NUVEM É O FORNECIMENTO DE SERVIÇOS DE COMPUTAÇÃO PELA INTERNET.
* PROPORCIONA INOVAÇÕES MAIS RÁPIDAS,FLEXÍVEIS E ECONÔMICAS EM ESCALA.
* ALGUNS DOS RECURSOS: REDES, ARMAZENAMENTO, PROCESSAMENTO DE DADOS, BANCO DE DADOS E SOFTWARES.

# QUAIS MODELOS DE NEGOCIO O AZURE TEM ?
* MODELO CAPEX (CAPITAL EXPENDITURE) "DESPESA DE CAPITAL" => É O MODELO ONDE A EMPRESA ARCA COM DESPESAS INCIAIS COMO:
  - GASTOS COM AQUISIÇÕES.
  - ESCALABILIDADE LIMITADA E DEPENDENTE DA INSTITUIÇÃO.
  - MELHORIAS OU MANUTENÇÃO COM ATIVOS AO LONGO PRAZO, COMO: COMPUTADORES, RACKS , PRÉDIOS, INFRAESTRUTURA EM GERAL.

* MODELO OPEX (Operational Expenditure) "DESPESA OPERACIONAL" => É O MODELO ONDE OS CUSTOS SÃO RECORRENTES PARA MANTER A OPERAÇÃO DA INSTITUIÇÃO ATIVA, COBRANÇA RECORRENTE CONFORME O USO DAS MAQUINAS DO AZURE, A ESCALABILIDADE É ELÁSTICA E AUTOMÁTICA, NÃO DEPENDE DA INSTITUIÇÃO.

# TIPOS DE NUVEM
* NUVEM PÚBLICA
  - SEM DESPESAS DE CAPITAL PARA ESCALAR VERTICALMENTE.
  - NÃO REQUER INVESTIMENTO INICIAL, CUSTO SERÁ POR USO.
  - UTILIZA DATACENTERS GLOBAIS.
  - ALTO GERENCIAMENTO PELO PROVEDOR.
  - BAIXO CONTROLE DA INFRA PELA INSTITUIÇÃO.

* NUVEM PRIVADA (ON-PREMISES)
  - UTILIZA A INFRAESTRUTURA LOCAL DA INSTITUIÇÃO.
  - CONTROLE TOTAL DA INFRAESTRUTURA E DADOS PELO INSTITUIÇÃO.
  - CUSTO POR USO.
  - MANUTENÇÃO FEITA TOTALMENTE PELO INSTITUIÇÃO.

* NUVEM HÍBRIDA
  - UTILIZA UM MISTO DE DE LOCAL + AZURE PUBLICA.
  - INSTITUIÇÃO DECIDE ONDE EXECUTAR SEUS APLICATIVOS .
  - INTITUIÇÃO CONTROLAM A SEGURANÇA, A CONFORMIDADE E OS REQUISITOS LEGAIS.
  - ACABA UTILIZANDO O MELHOR DOS DOIS CENÁRIOS ANTERIORES.  

# AZURE E COBRANÇAS
* MODELO BASEADO EM CONSUMO
  - QUANTO MAIOR A UTILIZAÇÃO, MAIOR A COBRANÇA.
  - ESSE MODELO TEM MAIOR PREVISÃO DE CUSTOS.
  - SÃO FORNECIDOS PREÇOS PARA SERVIÇOS E RECURSOS INDIVIDUAIS.
  - AZURE FORNECE SERVIÇOS PARA AUXILIAR A CÁLCULAR OS CUSTOS DA INSTIUIÇÃO, COMO A CALCULADORA.
